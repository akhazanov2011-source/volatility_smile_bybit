# WikiLLM Short: Volatility Smile Bybit

Проект строит `volatility smile` по опционам Bybit через публичный API V5 и работает только как веб-сервис. Вся Python-логика находится в `app.py`: Flask route'ы, загрузка тикеров, парсинг опционных символов, автоматический выбор страйкового окна, подготовка данных и построение Plotly-графика для `IV`, `theta` или `theta / mark price (%)`. `templates/index.html` — единственный UI-шаблон.

Что важно для LLM:

- В проекте остался один Python-файл: `app.py`.
- CLI-режима, Dash-режима и совместимого `volatility_smile.py` больше нет.
- Ключевая функция проекта — `fetch_and_prepare_data(...)`; она оставляет OTM-крылья и по одной ближайшей ATM/ITM точке.
- Диапазон страйков больше не задаётся вручную: он рассчитывается автоматически так, чтобы показать 70% ближайших к `spot` страйков.
- В интерфейсе есть переключатель метрики `iv` / `theta` / `theta_pct`.
- `build_figure(...)` ожидает уже подготовленную структуру `by_expiry`.
- Поддерживаемые монеты: `BTC`, `ETH`, `SOL`, `XRP`, `DOGE`, `XAUTUSDT`.
- Минимальная проверка после правок: `python3.14 -m py_compile app.py`, затем smoke-import `app`.
