Подсказка по графику

Hover по точкам показывает полный контекст опциона: выбранную метрику, IV, theta, delta, gamma, vega, а также высшие греки (vanna, volga, speed, charm, ultima), которые считаются локально по модели Блэка — Шоулза. Греки первого порядка берутся из API биржи там, где она их отдаёт (Bybit, Binance), иначе считаются по БС (Deribit, OKX). Безрисковая ставка для каждого опциона определяется автоматически из implied cost-of-carry (r = ln(forward/spot) / T) для бирж, отдающих forward (Bybit, OKX); для Deribit/Binance r = 0. Ставка выводится отдельной строкой в тултипе. Для каждой экспирации используется одна непрерывная линия.

Справочник греков

Метрики, доступные в выпадающем списке. API — значение берётся напрямую из ответа биржи (если она его отдаёт); BS — считается локально по модели Блэка — Шоулза (для бирж, не отдающих греки: Deribit/OKX, либо для высших греков на любой бирже); derived — нормировка данных API.
